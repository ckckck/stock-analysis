# 顶部栏文字按钮与 AI 筛选工作区布局设计

## 背景

当前顶部栏右上角的前两个入口仍然是纯图标，识别成本偏高；AI 筛选工作区的提示文案、输入框、结果条数选择和操作按钮也没有形成稳定的视觉对齐，导致操作区看起来松散。

## 目标

1. 将顶部栏前两个入口改成文字按钮，分别显示“龙虎榜”和“全网热点”。
2. 保持右侧主题切换与设置按钮现有样式不变。
3. 优化 AI 筛选工作区的对齐方式，让提示、输入框、结果条数选择、主按钮和“查看 SQL”按钮形成清晰的垂直节奏。
4. 将“输入自然语言，描述筛选方式”改成灰色提示文字。

## 方案

### 顶部栏

- 保留当前位置和点击行为不变。
- 将原来的两个图标按钮改成圆角文字按钮：
  - 龙虎榜
  - 全网热点
- 文字按钮沿用现有面板边框与 hover 风格，但增加水平内边距，保证与右侧图标按钮形成轻重对比。

### AI 筛选工作区

- 顶部提示文案改为灰色小标题，不再强调成深色主标题。
- 输入框保持单独一行，宽度占满。
- 操作区改成两层：
  - 第一层：左侧结果条数选择器，右侧主按钮
  - 第二层：仅在存在 SQL 时显示“查看 SQL”按钮，并与上方操作区整体居中对齐
- 主按钮与选择器的垂直对齐统一到同一基线，避免当前按钮高低不一的视觉问题。

## 影响范围

- `jcp/frontend/src/App.tsx`
- `jcp/frontend/src/components/ScreeningWorkspace.tsx`

## 验证

1. 顶部栏前两个按钮显示为文字按钮，主题和设置保持原样。
2. AI 筛选工作区提示文案为灰色。
3. 输入框、选择器、主按钮、“查看 SQL”按钮的对齐比当前更紧凑清晰。
